# Credit Risk Modeling

## Overview

This project predicts loan approval outcomes using machine learning techniques. The goal is to analyze applicant information and identify factors that influence lending decisions.

## Dataset

The project uses a Loan Prediction dataset containing applicant information such as:

- Gender
- Marital Status
- Education
- Applicant Income
- Coapplicant Income
- Loan Amount
- Credit History
- Property Area
- Loan Status

### Target Variable

| Value | Meaning |
|---------|---------|
| Y | Loan Approved |
| N | Loan Rejected |

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Key Insights

- Credit History is one of the strongest predictors of loan approval.
- Applicant Income significantly impacts lending decisions.
- Loan Amount and Property Area also influence approval outcomes.
